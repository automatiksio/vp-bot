# Varia Plus Landing Pages

Sales-optimierte Landing Pages für Varia Plus KI-Produkte.

## Produkte

### 1. KI-Anrufbeantworter (`/voice/`)
- **Ziel:** Anrufe automatisch entgegennehmen, Fragen beantworten, Termine buchen
- **Zielgruppe:** Handwerk, Praxen, kleine Unternehmen
- **Pakete:** Basic (99€), Standard (199€), Premium (349€)

### 2. Omni-Channel Chatbot (`/chatbot/`)
- **Ziel:** Kundenanfragen auf allen Kanälen automatisch beantworten
- **Zielgruppe:** Mittelstand mit mehreren Kontaktkanälen
- **Pakete:** Basic (79€), Standard (149€), Premium (279€)

## Deployment

Die Seiten sind statisch und können überall gehostet werden:
- Cloudflare Pages
- GitHub Pages
- Vercel
- Beliebiger Webserver

### Subdomain-Vorschlag:
- `voice.variaplus.de` → KI-Anrufbeantworter
- `chat.variaplus.de` → Omni-Channel Chatbot
- oder als Unterseiten: `variaplus.de/ki-anrufbeantworter/`, `variaplus.de/chatbot/`

## Design-System

- **Schriften:** Playfair Display (Headlines), Lato (Body)
- **Farben:**
  - Blau: #0064fa (Primary CTA)
  - Gold: #ffb204 (Akzent)
  - Dark: #111111 (Text)
  - Light: #f8f9fa (Hintergründe)
- **Style:** Seriös, nicht corporate; vertrauenswürdig aber modern

## Sales-Struktur (Eric Whitman Prinzipien)

1. **Hero** - Headline mit Hauptbenefit, starker CTA
2. **Problem** - Pain Points der Zielgruppe (emotional)
3. **Solution** - Produkt als Lösung positionieren
4. **Benefits** - Was der Kunde konkret bekommt
5. **How it Works** - 3 einfache Schritte (Komplexität wegnehmen)
6. **Social Proof** - Testimonials (Vertrauen aufbauen)
7. **Pricing** - Transparente Pakete (mittleres Paket hervorgehoben)
8. **FAQ** - Einwände vorwegnehmen
9. **Final CTA** - Letzte Handlungsaufforderung
